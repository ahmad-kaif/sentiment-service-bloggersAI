
### 1. Create a virtual env
```
 python3 -m venv venv
```

### 2. Activate the env
```
 source venv/bin/activate

```

### 3. Down load all the dependencies
```
pip install -r requirements.txt
```
### 4. Run
```
python3 app.py
```
